關於我!

你好！我是 @PanChing 我定義我自己是一個程式設計師. 我喜歡寫程式就像在生活中喜歡分析、解決問題的我一樣.這會給我帶來的成就感！
在我最後一個專案中負責公司眾多專案中其中一個項目：考勤系統. (上線專案) 
此項專案應使用 Java 為主要開發語言 而 資料庫 使用的是 MySql.用於銀行的內部資訊系統（ ex.出勤、薪資獎勵、報表...etc ).在這段過程中從中成長和學習到了許多.在閒暇之餘也嘗試用不同的語言開發了一些屬於自己的一些遊戲和作品.

知識與技術!

Java

在求學過程中有上過相關的ＶＢ及Ｃ語言的課程. 因而對程式語言產生興趣便開始自學程式.因為希望增加自己在這方面的知識能夠有系統的學習. 在畢業之後就到資策會上了相關的課程學習Ｗeb的知識技術. 而Orz互助平台 (未上線) 是我一個完整開發出專案. 主要語言為 Java 資料庫為 Oracle 在此專案中主要訴求為類似接案外包網的網站來替接案與發案者做媒合的動作.
專案名稱：Orz互助平台

Android programming

在 Android App 中 因為現在手機跟穿戴裝置的黏著度越來越高. 而我本身也是重症患者之一. 而在我的生活中因為時常因通勤的原因常會需要使用 台鐵官方網站 或是 別人的時刻表查詢的 App 來查詢火車時刻表. 這部分對我來說有點困擾.於是就決定自己動手寫一個自己的 查詢時刻表的 App 來解決我的需求.而同時也透過這個作品學習熟悉android的相關技術.所以我的第一個 App 就這樣誕生了！從開始撰寫的資料前處理、結構的設計 到 排版/使用者體驗、Logo 設計全部一首還真是有點累人Orz..
APP 名稱：Train
